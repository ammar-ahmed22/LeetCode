## Question
Given `n` non-negative integers representing an elevation map where the width of each bar is `1`, compute how much water it can trap after raining.

## Solution
```typescript
function trap(height: number[]): number {
    let l: number = 0;
    let r: number = height.length - 1;
    let lMax: number = height[l];
    let rMax: number = height[r];

    let result: number = 0;
    while(l < r){
        if (lMax <= rMax){
            l++
            lMax = Math.max(lMax, height[l]);
            result += lMax - height[l];
        } else {
            r--
            rMax = Math.max(rMax, height[r]);
            result += rMax - height[r];
        }
    }

    return result;
};
```

```python
class Solution:
    def trap(self, height: List[int]) -> int:
        l, r, result = 0, len(height) - 1, 0
        l_max = height[l]
        r_max = height[r]

        while l < r:
            if l_max <= r_max:
                l += 1
                l_max = max(l_max, height[l])
                result += l_max - height[l]
            else:
                r -= 1
                r_max = max(r_max, height[r])
                result += r_max - height[r]
        
        return result
```

## Explanation
- Initialize left and right pointers
- Initialize left and right max tracking variables
    * left and right max are set to the left and right side of the elevation map
- Iterate from both sides 
    * If left side is larger, increment left pointer
    * update left max
    * add the max of left subtract the current height to the result
    * otherwise do the same but for right side
- There can only be as much water trapped as the current highest point in the elevation which is the premise this solution follows
