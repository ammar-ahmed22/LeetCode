## Question
Given an array of integers `temperatures` represents the daily temperatures, return an array `answer` such that `answer[i]` is the number of days you have to wait after the `ith` day to get a warmer temperature. If there is no future day for which this is possible, keep `answer[i] == 0` instead.

## Solution
```typescript
function dailyTemperatures(temperatures: number[]): number[] {
    type StackEntry = {
        temp: number,
        index: number
    }
    const stack: StackEntry[] = [];
    const result: number[] = new Array(temperatures.length).fill(0);
    temperatures.forEach((t, i) => {
        while(!!stack.length && t > stack[stack.length - 1].temp){
            const { index } = stack.pop();
            result[index] = i - index;
        }
        stack.push({
            temp: t,
            index: i
        });
    })

    return result;
};
```

## Explanation
- Create a stack and result variable
- Iterate over the temperatures
- Add the temperature and index to the stack
- Iterate over the stack while it is not empty and the temperature at the top of the stack is less than the current temp
- Update the colder days with the number of days till the hotter day


## Complexity
**Runtime**: O(n) <br/>
**Space**: O(n)
